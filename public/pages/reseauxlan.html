IPv4 vs IPv6

IPv4 :

Format traditionnel (ex: 192.168.1.1). Limité à environ 4 milliards d'adresses uniques.

IPv6 :

Nouveau format (ex: 2001:0db8:85a3::8a2e:0370:7334). Pratiquement illimité en adresses.

Sous-réseaux :

Division d'un réseau en segments plus petits pour une meilleure organisation et sécurité.

Outils de Diagnostic

Commandes de base :

• ping : Tester la connectivité
• ipconfig/ifconfig : Configuration IP
• tracert/traceroute : Tracer le chemin réseau
• nslookup : Résolution DNS

Problèmes courants :

• Pas de connectivité Internet
• Lenteur réseau
• Problèmes DNS
• Conflits d'adresses IP
